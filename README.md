# CNN_Clasificador_Galaxias_DeepLearning
Este proyecto clasifica galaxias usando imágenes astronómicas y un modelo de IA. Tras entrenarlo, validarlo y refinarlo mediante análisis de errores, se obtuvo una herramienta precisa que reconoce tipos de galaxias en nuevas imágenes. Esto automatiza el proceso, acelerando la investigación y facilitando la divulgación científica.
